# Calculator Application (JavaFX)

This project is a simple Calculator Application developed using Java and JavaFX.
It provides a graphical user interface (GUI) for performing basic arithmetic operations.

## Features

- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)
- Clear / Reset functionality
- User-friendly JavaFX interface
- FXML-based UI design

## Technologies Used

- Java (JDK 8+)
- JavaFX
- FXML

## Project Structure

Calculator.java  
CalculatorFXML.fxml  
CalculatorFXMLController.java  
AdvanceJavaProject1.java  

## How to Run

1. Make sure Java JDK 8 or higher is installed.
2. Open the project in an IDE (IntelliJ IDEA, Eclipse, or NetBeans).
3. Ensure JavaFX is properly configured.
4. Run the `AdvanceJavaProject1.java` file.

If you are using Java 11 or higher, add JavaFX VM options when running the project.

## Notes

- This project is created for educational and practice purposes.
- The application follows a simple MVC-style structure using FXML.
